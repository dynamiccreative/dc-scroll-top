# DC Scroll Top

Plugin WordPress — bouton scroll to top personnalisable.

![downloads](https://img.shields.io/github/downloads/dynamiccreative/dc-scroll-top/total) ![downloads@latest](https://img.shields.io/github/downloads/dynamiccreative/dc-scroll-top/latest/total)

## Fonctionnalites

- **Activation / desactivation** sans desactiver le plugin (toggle on/off)
- **6 styles d'icones** SVG au choix
- **Couleur** personnalisable (color picker)
- **Position** configurable (bas et droite, en px)
- **Taille** du bouton ajustable
- **Animation** : Fade ou Slide
- **Distance d'apparition** : nombre de pixels scrolles avant d'afficher le bouton
- **Vitesse de remontee** : duree de l'animation de retour en haut
- **Responsive** : breakpoint mobile pour masquer le bouton
- **Preview live** dans l'admin pour visualiser les changements en temps reel
- **Accessibilite** : focus visible, title sur le lien, aria-compatible
- **Filtre PHP** `dc_scroll_top_show` pour controle programmatique

## Installation

1. Telecharger le ZIP depuis [Releases](https://github.com/dynamiccreative/dc-scroll-top/releases)
2. Dans WordPress : Extensions > Ajouter > Telecharger une extension
3. Activer le plugin
4. Configurer dans **DC Plus > Scroll Top** (ou Reglages > DC Scroll Top si DC Plus n'est pas installe)

## Mise a jour automatique

Le plugin se met a jour automatiquement via GitHub (integration GitHubUpdater).

## Requires

- WordPress 6.7+
- PHP 7.4+
